# dataset-shift
Package for detecting any dataset shift

Datasets usually face with three kinds of data shift issue: /n
    1. Covarance shift/ covariance drift /n
    2. Prior probability shift /n
    3. Concept shift/ Domain Shift /n
    4. Source Component shift /n
    5. Sample selection Bias /n
    
This package aims at detecting which variables carry the different shifts in data. [WIP]
